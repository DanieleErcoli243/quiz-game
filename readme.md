# Quiz-Game

## A game about questions and answers created with HTML, CSS and Javascript

This project is created by following a tutorial on FreeCodeCamp YouTube channel. It focuses on frontend skills, an user interface, with questions, score, answers and a progress bar made with pure languages. 


## How to use the project

In order to make it work, you basically have to clone the repo on your computer, open it with Visual Studio Code and then activate Live Server or open the repo with the browser you prefer.
